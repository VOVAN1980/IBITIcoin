# Contributing Guidelines

Thank you for your interest in IBITIcoin.

This repository is primarily maintained by the core development team.
External contributions are welcome but reviewed carefully.

---

## Scope of Contributions

Accepted:
- Bug fixes
- Gas optimizations
- Code clarity improvements
- Test coverage improvements (when tests are present)

Not accepted:
- Feature proposals without prior discussion
- Architectural rewrites
- Changes that alter token economics or supply logic

---

## Pull Requests

- Fork the repository
- Create a feature or fix branch
- Keep changes minimal and focused
- Add clear commit messages
- Open a Pull Request against `main`

All PRs are reviewed manually.
Approval is not guaranteed.

---

## Issues

Issues may be used for:
- Bug reports
- Clarification questions
- Documentation improvements

Security issues must follow `SECURITY.md` and **must not** be opened as public issues.

---

## Code Style

- Solidity `^0.8.x`
- Explicit visibility (`public`, `external`, `internal`)
- Clear naming over cleverness
- No hidden logic
